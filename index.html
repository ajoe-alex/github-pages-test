Projects

GET /projects
POST /projects/{projectId}/resources

Create a Project Resource

POST https://api.example.com/projects/{projectId}/resources?type=server

Create a new resource under an existing project. You can specify resource details and tags in the JSON body.

Request

Path Parameters

projectId string — ID of the project to which this resource will be added.

Query Parameters

type string — The resource type (e.g. server, database).
autoStart boolean — Whether the resource should auto-start on creation.

Request Body

Responses

201 Created 400 Bad Request

Response Body

Parameters

projectId* : string
type* : string
autoStart : boolean

Request Sample (cURL)

curl --request POST \
--url 'https://api.example.com/projects/123/resources?type=server&autoStart=true' \
--header 'Content-Type: application/json' \
--data '{
  "name": "App Server 1",
  "region": "us-east-1",
  "tags": ["production", "critical"]
}'
